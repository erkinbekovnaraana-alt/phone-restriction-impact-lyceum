# 📊 Phone Usage Impact Study

## Overview

This project explores the impact of phone access on student performance, focus, and sleep patterns in a structured academic environment.

## Objective

To analyze how a strict study environment without phones affects:

* Academic performance
* Focus level
* Sleep quality

## Methodology

* Created a dataset simulating student behavior
* Used Python (Pandas, Plotly) for analysis
* Compared students **with** and **without** phone access

## Key Findings

* Students without phone access showed **higher exam scores**
* Focus levels were significantly higher without phone distractions
* Slight decrease in sleep was observed in stricter environments

## Visualizations

* Phone Access vs Exam Score
* Phone Access vs Focus Level
* Phone Access vs Sleep Hours

## Conclusion

Reduced phone usage leads to improved concentration and better academic outcomes, highlighting the importance of minimizing distractions in learning environments.

## Technologies Used
